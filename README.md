# Página Web Invencível 🟡🔵⚫

Uma página estática dedicada à obra **Invencível (Invincible)**, criada como um projeto de estudo, prática de desenvolvimento web e apreciação de uma das HQs de super-heróis mais marcantes da era moderna.

O projeto apresenta informações sobre a história, personagens, referências, momentos importantes e uma visão pessoal sobre a obra de **Robert Kirkman**, utilizando a construção de uma página temática como forma de praticar fundamentos essenciais do desenvolvimento web.


[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Acessar%20Projeto-blue?style=for-the-badge&logo=github)](https://felipecorreia-ti.github.io/Invencible-WebSite/)





## 🎯 Objetivo do projeto

O principal objetivo deste projeto é praticar e aprimorar o uso dos **elementos semânticos do HTML**, desenvolvendo uma página organizada, estruturada e com uma divisão clara de conteúdo.

Durante o desenvolvimento, o foco esteve em compreender melhor a utilização de elementos como:

* `header` para organização do cabeçalho da página;
* `nav` para criação de navegação entre seções;
* `main` para estruturação do conteúdo principal;
* `section` para separação lógica dos assuntos;
* `article` para conteúdos independentes;
* `figure` e `figcaption` para associação entre imagens e descrições;
* `footer` para informações finais da página.

A ideia inicial foi construir uma página simples, priorizando uma boa estrutura HTML e a organização das informações antes de avançar para aspectos visuais e interativos.

## 🚀 Futuras melhorias

Este projeto continuará evoluindo com novas versões e melhorias, incluindo:

* Criação de layouts mais modernos e agradáveis;
* Aplicação de uma identidade visual inspirada no universo da obra;
* Melhor utilização de cores, espaçamentos e tipografia;
* Implementação de recursos utilizando CSS;
* Possíveis funcionalidades utilizando JavaScript;
* Melhorias na experiência de navegação e acessibilidade.

A proposta é transformar gradualmente uma página simples de estudo em uma experiência mais completa, mantendo como base uma estrutura HTML bem organizada.

## 📖 Sobre o projeto

A página reúne:

* Um resumo geral da HQ *Invencível*;
* Informações sobre sua criação e seus autores;
* Referências e inspirações presentes na obra;
* Destaques de momentos marcantes da história;
* Uma análise dos principais temas abordados;
* Uma opinião pessoal como leitor e admirador das obras de Robert Kirkman.

Mais do que apenas apresentar informações, o projeto busca demonstrar como uma obra de super-heróis pode explorar temas como **responsabilidade, legado, escolhas, família e consequências**.

## 🌎 Sobre a obra

*Invencível* acompanha a jornada de **Mark Grayson**, um jovem que descobre possuir poderes extraordinários e precisa lidar com as responsabilidades e conflitos que surgem junto deles.

Criada por **Robert Kirkman**, **Cory Walker** e posteriormente desenvolvida com a arte de **Ryan Ottley**, a HQ foi publicada pela Image Comics entre 2003 e 2018, totalizando 144 edições.

A obra mistura ação, ficção científica, drama e política, construindo um universo onde decisões possuem consequências permanentes e os personagens evoluem ao longo da narrativa.

## 📚 Contexto de aprendizado

Este repositório representa uma etapa inicial no aprendizado de desenvolvimento web, utilizando um tema de interesse pessoal como motivação para praticar conceitos fundamentais.

A escolha de *Invencível* como tema permite unir estudo e criatividade, tornando o processo de aprendizado mais significativo enquanto novas tecnologias e melhorias são adicionadas ao projeto.

## ⚠️ Aviso

A página contém comentários sobre acontecimentos importantes da HQ, incluindo possíveis **spoilers** da história.

Caso ainda não tenha lido *Invencível*, recomenda-se conhecer a obra antes de explorar todas as seções.
